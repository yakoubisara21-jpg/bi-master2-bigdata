TP BI – Talend  
Master 2 Big Data Analytics – USTHB  

Ce dossier contient les éléments du TP réalisé avec Talend pour la création d’un entrepôt de données (Data Warehouse) basé sur la base Northwind utilisée sous SQL Server 2022.

-----------------------------------------
1. Contenu du dossier
-----------------------------------------

• diagrammes/
  Contient les schémas (diagrammes) liés au projet :
  - Diagramme relationnel de Northwind
  - Diagramme relationnel de l’entrepôt DataWHCommande

• rapports/
  Contient les rapports en PDF :
  - Rapport de création de la base Northwind
  - Rapport de création de l’entrepôt DataWHCommande
  - Explications détaillées des étapes Talend

• sql/
  Contient les scripts SQL utilisés :
  - Script de création des tables de l’entrepôt (Dim_Client, Dim_Employee, Dim_Temps, TF_Commande)
  - Script de création de la base DataWHCommande
  - Requête d’analyse (Commandes Livrées / Non Livrées)

-----------------------------------------
2. Objectif du TP
-----------------------------------------

Le but du TP est de :
• se connecter à SQL Server 2022 avec Talend (compte : sa / sa@123@123)  
• extraire les données des tables Northwind  
• les transformer dans Talend (calculs, regroupements, dates…)  
• charger les dimensions et la table de faits dans l’entrepôt DataWHCommande  
• produire un diagramme de l’entrepôt et des rapports PDF

-----------------------------------------
3. Technologies utilisées
-----------------------------------------

• SQL Server Management Studio 2022 (SSMS)
• Talend Open Studio for Data Integration
• LaTeX (rédaction des rapports)
• GitHub (organisation du projet)
